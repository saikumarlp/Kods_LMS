# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of the KosLearn LMS platform very seriously.

If you discover any security-related issues, please do NOT create a public GitHub issue. Instead, please privately email the repository owner or reach out through secure channels. 

Please provide the following details:
1. Steps to reproduce the vulnerability.
2. The potential impact of the vulnerability.
3. Any suggested mitigations.

We will endeavor to respond to your report within 48 hours and work with you to understand and patch the vulnerability promptly. We will credit you in the release notes if you wish.

Thank you for helping to keep our project secure!
